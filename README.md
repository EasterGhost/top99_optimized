# 拓扑优化 99 行代码优化版 (top99_optimized)

## 拓扑优化的基本概念

拓扑优化（Topology Optimization）是一种结构优化的数学方法，旨在给定的设计空间内，在特定的载荷、边界条件和制造约束下，优化材料的分布情况，以使系统的宏观性能达到最优。与传统的尺寸优化和形状优化相比，拓扑优化不依赖于现有的设计初始构型，能够在设计空间中自由地生成孔洞或改变材料的连通性，从而获得更轻巧且性能更优异的新颖结构。

## 优化模型

本仓库的算法基于经典的**变密度法 (SIMP, Solid Isotropic Material with Penalization)**。该模型的基本数学表述为：

- **目标函数 (Objective Function)**：结构柔顺度（即应变能）最小化，等价于结构刚度最大化。
- **设计变量 (Design Variables)**：有限元网格中每个单元的伪密度 $x$，取值范围连续分布在 0（无材料/孔洞）到 1（实体材料）之间。
- **约束条件 (Constraints)**：
  - **体积约束 (Volume Constraint)**：优化后结构的总材料体积不得超过给定的目标设定体积。
  - **平衡方程 (Equilibrium Equation)**：结构的受力和变形须满足有限元分析的静态平衡方程 $\mathbf{K}\mathbf{U} = \mathbf{F}$。

通过引入材料惩罚因子（通常取$penal=e$），SIMP 方法能够迫使中间密度单元向 0 和 1 两极分化，从而得到具有清晰边界的实体结构拓扑。

## 本仓库的内容

本仓库基于经典的 99 行 MATLAB 拓扑优化程序进行改进，提升了易读性和执行效率。主要包含以下文件：

- README.md：本项目的基本说明文档。
- LICENCE：本仓库及代码的开源授权协议文件。
- top99_optimized.m：经过改进和优化的 99 行拓扑优化 MATLAB 源代码。
- top99_optimized_Chinese.m：包含详尽中文注释的拓扑优化代码版本，方便中文语境下的初学者快速入门，理解算法流程与具体实现细节。
